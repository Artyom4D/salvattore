How it works?

Salvattore organizes your HTML elements according to the number of columns you specify. Each of the items in your container is placed within these columns, one by one. To get started simply add a data-columns attribute to the container.

<div id="timeline" data-columns>
  <div>Item #1</div>
  <div>Item #2</div>
  <div>Item #3</div>
  <div>Item #4</div>
  <div>Item #5</div>
</div>

In your CSS file you can set the number of columns you want to be created inside that element. To create a responsive columnar design, just use media queries!

[data-columns]::before { display: none; } /* Don't forget this. */

#timeline[data-columns]::before {
    content: '2';
}

@media screen and (min-width: 601px){
  #timeline[data-columns]::before {
    content: '3';
  }
}

Use the content property to tell Salvattore how many columns to generate. The script will distribute the original container's child elements automatically into the newly generated columns.

<div id="timeline" data-columns>
  <div class="column">
    <div>Item #1</div>
    <div>Item #4</div>
  </div>
  <div class="column">
    <div>Item #2</div>
    <div>Item #5</div>
  </div>
  <div class="column">
    <div>Item #3</div>
  </div>
</div>

Requirements

None. Salvattore is a standalone JS file, it will run automatically once called in the HTML document. It does not use jQuery to work. However, you can still use jQuery or any other JS library to work with the API to get advanced functionality.

Customization

The default class for each column is .column, but you can re-define it or even use more than one so that you have a flexible and modular code.

#timeline[data-columns]::before {
  content: '2 .col.one-half';
}

@media screen and (min-width: 601px){
  #timeline[data-columns]::before {
    content: '3 .col.one-third';
  }
}

Dynamic content

Salvattore has two functions you can work with: append_elements and prepend_elements. These two functions have the same arguments: the grid and the elements you want added to it.

var grid = document.querySelector('#timeline');
var item = document.createElement('article');

salvattore['append_elements'](grid, [item])
item.outerHTML = 'I’ve been appended!';
