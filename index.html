How it works?

Simply add a data-columns attribute to the element that contains the blocks you want on your layout. All the settings and conditions are defined in CSS.

<div id="timeline" data-columns>
  <div>Item #1</div>
  <div>Item #2</div>
  <div>Item #3</div>
  <div>Item #4</div>
  <div>Item #5</div>
</div>

In your CSS file you can set the number of columns you want to be created inside that element. You can even use media queries.

#timeline[data-columns] {
    content: '2';
}

@media screen and (min-width: 601px){
  #timeline[data-columns] {
    content: '3';
  }
}

Use the content property as a wildcard to tell Salvattore how many columns you want. The script will create columns as elements where each of the original elementâ€™s items will be placed transversely in order until all of them have a column.

<div id="timeline" data-columns>
  <div class="column">
    <div>Item #001</div>
    <div>Item #004</div>
  </div>
  <div class="column">
    <div>Item #002</div>
    <div>Item #005</div>
  </div>
  <div class="column">
    <div>Item #003</div>
  </div>
</div>

Customization

The class for each column is .column by default, but you can also re-define it or even use more than one through CSS so that you have a flexible and modular code.

#timeline[data-columns] {
  content: '2 .col.one-half';
}

@media screen and (min-width: 601px){
  #timeline[data-columns] {
    content: '3 .col.one-third';
  }
}
